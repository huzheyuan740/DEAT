# DEAT
DRL-based environment-adaptive offloading framework
## Usage:
To train DDPG:
```bash
python main.py --algorithm ddpg
```
To train MADDPG:
```bash
python main.py --algorithm maddpg
```
To train DC-DRL:
```bash
python main.py --algorithm dc_drl
```
To train DEAT:
```bash
python main.py --algorithm deat
```
